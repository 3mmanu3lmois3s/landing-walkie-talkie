¿Qué es Teethlion?

Teethlion es una red de comunicación local descentralizada que funciona sin Internet. Ideal para hogares, escuelas, instituciones y gobiernos.

  • Comunicación de voz en tiempo real sobre LAN, MAN, WAN y GAN.
  • Sin necesidad de servidores centrales.
  • Seguridad mediante cifrado de grado militar.
  • Ideal para emergencias, contingencias y soberanía tecnológica.
  • Software libre para personas y escuelas, con licencias especiales para gobiernos e instituciones.

🧠 Preguntas Frecuentes (FAQ)

¿Qué necesito para usar Teethlion? +

Solo dispositivos conectados a la misma red LAN. No necesitas internet, servidores ni routers especiales.

¿Qué pasa si alguien viola la licencia? +

Las licencias AGPL y TOL son legalmente vinculantes. El uso no autorizado puede enfrentar acciones legales.

¿Por qué es gratis para algunos y de pago para otros? +

Promovemos acceso libre para educación y hogares. Las licencias comerciales financian el desarrollo.
